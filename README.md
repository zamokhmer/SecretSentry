## üì¶ What is SecretSentry?

SecretSentry is a local security scanner designed for your Home Assistant configurations. It checks your configuration directory for potential security threats, including exposed credentials and insecure settings. By using SecretSentry, you can ensure your setup is safer and follows best practices.
